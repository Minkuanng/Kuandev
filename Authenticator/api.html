Giới thiệu

API này hỗ trợ đại lý / website đối tác để:

  • 📦 Lấy danh sách sản phẩm từ kho hàng (GET)
  • 🛒 Đặt hàng tự động (POST)
  • 📥 Nhập dữ liệu sản phẩm vào kho (POST)
  • 💰 Kiểm tra số dư đại lý (GET)

Base URL: https://vercel-api-mocha-zeta.vercel.app/api

⚠️ Base URL này là ví dụ. Bạn cần deploy repo vercel-api để có URL thực tế.

Cài đặt & Deploy

1
Clone repository
git clone https://github.com/Minkuanng/vercel-api.git
2
Cài đặt dependencies
npm install
3
Deploy lên Vercel
Import repository lên Vercel → Deploy
4
Thêm biến môi trường (Environment Variables)
FIREBASE_PROJECT_ID · FIREBASE_CLIENT_EMAIL · FIREBASE_PRIVATE_KEY · FIREBASE_DATABASE_URL
Lấy Firebase Admin credentials: Vào Firebase Console → Cài đặt → Tài khoản dịch vụ → Tạo khóa riêng tư mới. Copy JSON và lấy các trường tương ứng.
Quan trọng: FIREBASE_PRIVATE_KEY cần được escape đúng cách. Trên Vercel, thay \n thành \\n.

📥 POST /api/products - Nhập sản phẩm vào kho

POST https://vercel-api-mocha-zeta.vercel.app/api/products

Dùng để: Nhập dữ liệu sản phẩm từ bên ngoài vào kho hàng Firebase.

📤 Body (JSON)

TrườngKiểuBắt buộcMô tả
namestring✔Tên sản phẩm
pricenumber✔Giá bán (VND)
categorystring✔ID danh mục (lấy từ Firebase)
datastring hoặc array✔Dữ liệu sản phẩm, mỗi dòng 1 món
iconstringIcon emoji (mặc định 📦)
imagestringURL hình ảnh
descriptionstringMô tả sản phẩm

Ví dụ cURL - Nhập 1 sản phẩm

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acc Free Fire VIP",
    "price": 25000,
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "icon": "🔥",
    "image": "https://example.com/ff.jpg",
    "description": "Acc Free Fire VIP - 100k diamonds",
    "data": "acc1:pass1\nacc2:pass2\nacc3:pass3"
  }'

✅ Response thành công

{
  "success": true,
  "message": "Product created successfully",
  "productId": "-Nabc123xyz",
  "data": {
    "id": "-Nabc123xyz",
    "name": "Acc Free Fire VIP",
    "price": 25000,
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "icon": "🔥",
    "image": "https://example.com/ff.jpg",
    "description": "Acc Free Fire VIP - 100k diamonds",
    "data": "acc1:pass1\nacc2:pass2\nacc3:pass3",
    "sold": 0,
    "createdAt": 1721059200000
  }
}

📥 POST /api/products/batch - Nhập nhiều sản phẩm

POST https://vercel-api-mocha-zeta.vercel.app/api/products/batch

Dùng để: Nhập nhiều sản phẩm cùng lúc, tiết kiệm thời gian.

📤 Body (JSON)

TrườngKiểuBắt buộcMô tả
categorystring✔ID danh mục cho tất cả sản phẩm
productsarray✔Mảng các sản phẩm

Ví dụ cURL - Nhập 2 sản phẩm cùng lúc

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/products/batch \
  -H "Content-Type: application/json" \
  -d '{
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "products": [
      {
        "name": "Acc Free Fire VIP",
        "price": 25000,
        "icon": "🔥",
        "data": "acc1:pass1\nacc2:pass2"
      },
      {
        "name": "Acc Liên Quân",
        "price": 15000,
        "icon": "⚔️",
        "data": "data1\ndata2\ndata3"
      }
    ]
  }'

✅ Response thành công

{
  "success": true,
  "message": "Imported 2 products, 0 failed",
  "data": {
    "results": [
      { "id": "-Nabc123xyz", "name": "Acc Free Fire VIP" },
      { "id": "-Ndef456uvw", "name": "Acc Liên Quân" }
    ],
    "errors": []
  }
}

🛒 POST /api/order - Đặt hàng

POST https://vercel-api-mocha-zeta.vercel.app/api/order

📤 Body (JSON)

TrườngKiểuBắt buộcMô tả
tokenstring✔Token đại lý
shop_idstring✔Mã đại lý
product_idstring✔ID sản phẩm
quantitynumberSố lượng (mặc định 1)
input_datastringThông tin đối tác

Ví dụ cURL

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/order \
  -H "Content-Type: application/json" \
  -d '{
    "token": "sk_xxx...",
    "shop_id": "shop_ab12cd3",
    "product_id": "-Nabc123xyz",
    "quantity": 1,
    "input_data": "user123@game"
  }'

✅ Response thành công

{
  "success": true,
  "order_code": "#A1B2C3",
  "product_name": "Acc Free Fire VIP",
  "quantity": 1,
  "price": 25000,
  "balance_left": 475000,
  "data": ["acc1:pass1", "acc2:pass2"]
}

❌ Mã lỗi thường gặp

HTTPerrorÝ nghĩa
401invalid_tokenToken sai hoặc không khớp shop_id
403shop_bannedĐại lý bị khóa
404product_not_foundSản phẩm không tồn tại
409out_of_stockHết hàng
402insufficient_balanceSố dư không đủ

Nhập kho từ API bên ngoài

Bạn có thể lấy dữ liệu sản phẩm từ API bên ngoài và nhập trực tiếp vào kho hàng bằng tính năng "API Import" trong tab API Import của trang quản trị.

1. Lấy JSON 2. Paste URL 3. Chọn mục 4. Nhập kho

📥 Định dạng JSON hỗ trợ

FormatVí dụ
{ "products": [...] }API của bạn
{ "data": [...] }API khác
[ ... ]Mảng trực tiếp
{ "success": true, "products": [...] }Vercel API
Lưu ý: Khi import, hệ thống chỉ lấy các trường name, price, data (dạng string, mỗi dòng 1 món). Các trường khác như icon, image, description là tùy chọn.

Tóm tắt luồng làm việc

1️⃣ Deploy
Deploy vercel-api lên Vercel
2️⃣ Tạo đại lý
Tạo shop_id + token trong Admin
3️⃣ POST nhập kho
Gửi dữ liệu đến /api/products
4️⃣ POST đặt hàng
Gửi request đến /api/order
Hoàn tất! Đại lý có thể đặt hàng tự động qua API, admin quản lý số dư và kho hàng qua Firebase.
