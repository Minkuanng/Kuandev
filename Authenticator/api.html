API Đại lý
Về trang quản trị

Tài liệu API đặt hàng

API cho phép website đối tác (đại lý) đặt mua sản phẩm trực tiếp bằng mã shop_id + token được cấp trong trang quản trị (tab API / Đại lý). Mỗi đại lý có số dư riêng, KHÔNG dùng chung ví người dùng trên web.

Xác thực

Mỗi request cần gửi kèm shop_id và token tương ứng (lấy trong trang quản trị). Token là bí mật — không chia sẻ công khai, không nhúng vào code phía trình duyệt (frontend) của web đối tác. Chỉ gọi API này từ server của đối tác.

Nếu lộ token, vào tab API / Đại lý → Sửa → bấm nút xoay vòng để tạo token mới ngay lập tức.

Đặt hàng

POST https://asia-southeast1-shop-c6777.cloudfunctions.net/api/order

Body (JSON)

TrườngKiểuBắt buộcMô tả
tokenstring✔Token của đại lý
shop_idstring✔Mã đại lý
product_idstring✔ID sản phẩm (xem trong tab Kho hàng, hoặc gọi GET /product/:id)
quantitynumberSố lượng, mặc định 1
input_datastringThông tin do đối tác truyền vào (VD: UID người nhận, ghi chú đơn) — được lưu lại kèm đơn hàng để đối chiếu

Ví dụ request

curl -X POST https://asia-southeast1-shop-c6777.cloudfunctions.net/api/order \
  -H "Content-Type: application/json" \
  -d '{
    "token": "sk_xxx...",
    "shop_id": "shop_ab12cd3",
    "product_id": "-Nabc123xyz",
    "quantity": 1,
    "input_data": "user123@game"
  }'

Response thành công (200)

{
  "success": true,
  "order_code": "#A1B2C3",
  "product_id": "-Nabc123xyz",
  "product_name": "Acc Free Fire VIP",
  "quantity": 1,
  "price": 25000,
  "balance_left": 475000,
  "data": ["acc1:pass1"]
}

Kiểm tra sản phẩm

GET https://asia-southeast1-shop-c6777.cloudfunctions.net/api/product/:product_id

Dùng để xem giá và tồn kho trước khi đặt hàng.

{
  "success": true,
  "product_id": "-Nabc123xyz",
  "name": "Acc Free Fire VIP",
  "price": 25000,
  "stock": 14
}

Kiểm tra số dư đại lý

GET https://asia-southeast1-shop-c6777.cloudfunctions.net/api/shop/balance?shop_id=...&token=...
{
  "success": true,
  "shop_id": "shop_ab12cd3",
  "balance": 475000,
  "status": "active"
}

Ví dụ gọi bằng JavaScript (Node.js)

const res = await fetch("https://asia-southeast1-shop-c6777.cloudfunctions.net/api/order", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    token: "sk_xxx...",
    shop_id: "shop_ab12cd3",
    product_id: "-Nabc123xyz",
    quantity: 1,
    input_data: "user123@game"
  })
});
const data = await res.json();
console.log(data);
Luôn gọi API này từ backend của đối tác, không gọi trực tiếp từ trình duyệt của khách hàng cuối vì token sẽ bị lộ.
