Quản trị

Tài liệu API đặt hàng

API dành cho website đối tác (đại lý) đặt mua sản phẩm bằng mã shop_id + token được cấp trong trang quản trị (tab API / Đại lý). Mỗi đại lý có số dư riêng, không dùng chung ví người dùng.

Xác thực

Mỗi request cần gửi kèm shop_id và token tương ứng (lấy trong trang quản trị). Token là bí mật — không chia sẻ công khai, không nhúng vào code phía trình duyệt. Chỉ gọi API từ server của đối tác.

Nếu lộ token, vào tab API / Đại lý → Sửa → bấm nút xoay vòng để tạo token mới ngay lập tức.

Đặt hàng

POST https://vercel-api-mocha-zeta.vercel.app/api/order

Body (JSON)

Trường Kiểu Bắt buộc Mô tả
tokenstring✔Token của đại lý
shop_idstring✔Mã đại lý
product_idstring✔ID sản phẩm (xem tab Kho hàng hoặc GET /product/:id)
quantitynumberSố lượng, mặc định 1
input_datastringThông tin đối tác truyền vào (VD: UID, ghi chú) — lưu kèm đơn

Ví dụ request

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/order \
  -H "Content-Type: application/json" \
  -d '{
    "token": "sk_xxx...",
    "shop_id": "shop_ab12cd3",
    "product_id": "-Nabc123xyz",
    "quantity": 1,
    "input_data": "user123@game"
  }'

Response thành công (200)

{
  "success": true,
  "order_code": "#A1B2C3",
  "product_id": "-Nabc123xyz",
  "product_name": "Acc Free Fire VIP",
  "quantity": 1,
  "price": 25000,
  "balance_left": 475000,
  "data": ["acc1:pass1"]
}

Kiểm tra sản phẩm

GET https://vercel-api-mocha-zeta.vercel.app/api/product/:product_id

Xem giá và tồn kho trước khi đặt hàng.

{
  "success": true,
  "product_id": "-Nabc123xyz",
  "name": "Acc Free Fire VIP",
  "price": 25000,
  "stock": 14
}

Kiểm tra số dư đại lý

GET https://vercel-api-mocha-zeta.vercel.app/api/shop/balance?shop_id=...&token=... 
{
  "success": true,
  "shop_id": "shop_ab12cd3",
  "balance": 475000,
  "status": "active"
}

Mã lỗi

HTTP error Ý nghĩa
400missing_fieldsThiếu trường bắt buộc
400invalid_quantityquantity không hợp lệ
401invalid_tokenToken sai hoặc không khớp shop_id
403shop_bannedĐại lý đã bị khóa
404shop_not_foundshop_id không tồn tại
404product_not_foundproduct_id không tồn tại
409out_of_stockKhông đủ hàng trong kho
402insufficient_balanceSố dư đại lý không đủ
500server_errorLỗi máy chủ, thử lại sau

Ví dụ gọi bằng JavaScript (Node.js)

const res = await fetch("https://vercel-api-mocha-zeta.vercel.app/api/order", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    token: "sk_xxx...",
    shop_id: "shop_ab12cd3",
    product_id: "-Nabc123xyz",
    quantity: 1,
    input_data: "user123@game"
  })
});
const data = await res.json();
console.log(data);
Luôn gọi API từ backend của đối tác, không gọi trực tiếp từ trình duyệt khách hàng cuối vì token sẽ bị lộ.
