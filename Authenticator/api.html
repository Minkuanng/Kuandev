Giới thiệu

API này hỗ trợ đại lý / website đối tác để:

  • 📦 Lấy danh sách sản phẩm từ kho hàng (GET)
  • 📥 Nhập dữ liệu sản phẩm vào kho (POST)
  • 🛒 Đặt hàng tự động (POST)
  • 💰 Kiểm tra số dư đại lý (GET)

Base URL: https://vercel-api-mocha-zeta.vercel.app/api

⚠️ Thay vercel-api-mocha-zeta.vercel.app bằng domain thực tế của bạn sau khi deploy.

API đang hoạt động

Các endpoint dưới đây đã được kiểm tra và hoạt động ổn định.

📥 POST /api/products - Nhập sản phẩm

POST https://vercel-api-mocha-zeta.vercel.app/api/products

Dùng để: Nhập một sản phẩm mới vào kho hàng Firebase.

📤 Body (JSON)

TrườngKiểuBắt buộcMô tả
namestring✔Tên sản phẩm
pricenumber✔Giá bán (VND)
categorystring✔ID danh mục (lấy từ Firebase)
datastring✔Dữ liệu sản phẩm, mỗi dòng 1 món
iconstringIcon emoji (mặc định 📦)
imagestringURL hình ảnh
descriptionstringMô tả sản phẩm

Ví dụ cURL

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acc Free Fire VIP",
    "price": 25000,
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "icon": "🔥",
    "image": "https://example.com/ff.jpg",
    "description": "Acc Free Fire VIP - 100k diamonds",
    "data": "acc1:pass1\nacc2:pass2\nacc3:pass3"
  }'

✅ Response thành công

{
  "success": true,
  "message": "Product created successfully",
  "productId": "-Nabc123xyz",
  "data": {
    "id": "-Nabc123xyz",
    "name": "Acc Free Fire VIP",
    "price": 25000,
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "icon": "🔥",
    "image": "https://example.com/ff.jpg",
    "description": "Acc Free Fire VIP - 100k diamonds",
    "data": "acc1:pass1\nacc2:pass2\nacc3:pass3",
    "sold": 0,
    "createdAt": 1721059200000
  }
}

📥 POST /api/products/batch - Nhập nhiều sản phẩm

POST https://vercel-api-mocha-zeta.vercel.app/api/products/batch

Dùng để: Nhập nhiều sản phẩm cùng lúc, tiết kiệm thời gian.

📤 Body (JSON)

TrườngKiểuBắt buộcMô tả
categorystring✔ID danh mục cho tất cả sản phẩm
productsarray✔Mảng các sản phẩm

Ví dụ cURL

curl -X POST https://vercel-api-mocha-zeta.vercel.app/api/products/batch \
  -H "Content-Type: application/json" \
  -d '{
    "category": "-NkGgxTsjL1Kb3R2XcF",
    "products": [
      {
        "name": "Acc Free Fire VIP",
        "price": 25000,
        "icon": "🔥",
        "data": "acc1:pass1\nacc2:pass2"
      },
      {
        "name": "Acc Liên Quân",
        "price": 15000,
        "icon": "⚔️",
        "data": "data1\ndata2\ndata3"
      }
    ]
  }'

✅ Response thành công

{
  "success": true,
  "message": "Imported 2 products, 0 failed",
  "data": {
    "results": [
      { "id": "-Nabc123xyz", "name": "Acc Free Fire VIP" },
      { "id": "-Ndef456uvw", "name": "Acc Liên Quân" }
    ],
    "errors": []
  }
}

📦 GET /api/products - Lấy danh sách sản phẩm

GET https://vercel-api-mocha-zeta.vercel.app/api/products

Dùng để: Lấy danh sách tất cả sản phẩm trong kho.

Tham sốKiểuMô tả
categorystringLọc sản phẩm theo danh mục
limitnumberSố lượng sản phẩm tối đa (mặc định 50)

curl https://vercel-api-mocha-zeta.vercel.app/api/products?limit=10

  "count": 10,
  "data": [

      "name": "Acc Free Fire VIP",

      "icon": "🔥",
      "image": "https://example.com/ff.jpg",
      "description": "Acc Free Fire VIP - 100k diamonds",
      "data": "acc1:pass1\nacc2:pass2\nacc3:pass3",
      "sold": 0,
      "createdAt": 1721059200000
    }
  ]
}

Hướng dẫn cài đặt

1
Clone repository
git clone https://github.com/Minkuanng/vercel-api.git
2
Cài đặt dependencies
npm install
3
Deploy lên Vercel
Import repository lên Vercel → Deploy
4
Thêm biến môi trường
FIREBASE_PROJECT_ID · FIREBASE_CLIENT_EMAIL · FIREBASE_PRIVATE_KEY · FIREBASE_DATABASE_URL
Lấy Firebase Admin credentials: Vào Firebase Console → Cài đặt → Tài khoản dịch vụ → Tạo khóa riêng tư mới. Copy JSON và lấy các trường tương ứng.
Quan trọng: FIREBASE_PRIVATE_KEY cần được escape đúng cách. Trên Vercel, thay \n thành \\n.
