Cấu hình API Gateway 🔴 Chưa kết nối
Lưu ý: GitHub Pages không hỗ trợ API server-side. Sử dụng Vercel API để có endpoint thật.
Nhập dữ liệu sản phẩm từ API Hỗ trợ JSON từ Vercel / API bên ngoài

Hướng dẫn sử dụng API Import

1. Gửi dữ liệu từ bên ngoài vào: Sử dụng endpoint POST /api/products hoặc POST /api/products-batch

2. Định dạng JSON:

{
  "products": [
    {
      "name": "Tên sản phẩm",
      "price": 100000,
      "icon": "🎮",
      "image": "https://example.com/image.jpg",
      "description": "Mô tả sản phẩm",
      "data": "item1\nitem2\nitem3"
    }
  ]
}

3. Ví dụ gọi API bằng cURL:

curl -X POST https://vercel-api-minkuanngs-projects.vercel.app/api/products-batch \
  -H "Content-Type: application/json" \
  -d '{
    "category": "CATEGORY_ID_HERE",
    "products": [
      {
        "name": "Acc Game 1",
        "price": 50000,
        "data": "acc1:pass1\nacc2:pass2"
      }
    ]
  }'

4. Hoặc sử dụng JavaScript:

fetch('https://vercel-api-minkuanngs-projects.vercel.app/api/products-batch', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    category: 'CATEGORY_ID_HERE',
    products: [
      { name: 'Acc Game', price: 50000, data: 'acc1:pass1\nacc2:pass2' }
    ]
  })
})
.then(res => res.json())
.then(data => console.log(data));
Lưu ý: API đã được deploy sẵn tại vercel-api-minkuanngs-projects.vercel.app (repo GitHub Minkuanng/vercel-api), không cần tự dựng thêm.
