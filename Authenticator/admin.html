Nhập sản phẩm từ API

Hướng dẫn API Import

1. Gửi dữ liệu từ bên ngoài: POST /api/products/batch

2. Định dạng JSON:

{
  "category": "CATEGORY_ID",
  "products": [
    {
      "name": "Tên SP",
      "price": 100000,
      "icon": "🎮",
      "image": "https://...",
      "description": "Mô tả",
      "data": "item1\nitem2\nitem3"
    }
  ]
}
Deploy vercel-api lên Vercel để có endpoint thật. Xem repo github.com/Minkuanng/vercel-api
